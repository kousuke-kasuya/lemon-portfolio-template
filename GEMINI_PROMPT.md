# Gemini CLI 使い方ガイド

Gemini CLI を使ってポートフォリオに機能を追加しよう！

---

## 基本的な使い方

ターミナルで以下のように入力します：

```
gemini
```

起動したら、やりたいことを日本語で伝えてください。

---

## コピペで使える指示例

### スクロールアニメーション

```
js/main.jsに、スクロールしたときに各セクションがふわっと表示される
アニメーションを追加して。
HTMLやCSSは変えないで、JavaScriptだけで実装して。
IntersectionObserver を使ってね。
```

---

### ハンバーガーメニュー（スマホ用）

```
スマホ用のハンバーガーメニューを追加したい。
index.htmlのheaderの .nav__menu の前にボタンを追加して、
js/main.jsに開閉の処理を書いて。
css/style.cssにもスタイルを追加して。
スマホ（768px以下）のときだけボタンが表示されるようにして。
```

---

### ダークモード切り替え

```
ページ右下にダークモード切り替えボタンを追加して。
css/style.cssの :root にあるカスタムプロパティを使って
色を切り替える方式にして。
ボタンは固定表示（fixed）で、アイコンは🌙と☀️を使ってOK。
```

---

### タイピングアニメーション

```
index.htmlのheroセクションにある .hero__tagline の
テキストを、1文字ずつタイピングされるように見せる
アニメーションをjs/main.jsに追加して。
ループはしなくていい。最初の1回だけ再生してね。
```

---

### Works カードのモーダル表示

```
index.htmlのWorksセクションのカードをクリックすると
詳細がモーダルで表示される機能を追加して。
モーダルの中にはカードのタイトル・説明・タグを表示してね。
閉じるボタンとオーバーレイクリックでも閉じられるようにして。
```

---

### ページトップに戻るボタン

```
ページ右下に「ページトップに戻る」ボタンを追加して。
スクロールが300px以上になったら表示して、それ以下では非表示にして。
クリックするとスムーズにトップに戻るようにして。
```

---

## うまくいかないときは

- 一度に複数のことを頼まず、1つずつお願いする
- 「〇〇ファイルの〇〇の部分に追加して」と場所を具体的に伝える
- エラーが出たらエラー文をそのままコピペして「このエラーを直して」と伝える
- 修正がうまくいかない場合は「変更を元に戻して」と伝える
